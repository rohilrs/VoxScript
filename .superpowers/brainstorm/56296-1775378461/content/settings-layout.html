Settings Page Layout

Two sections: General (interactive controls) and System (toggles). How should these be organized?

A

Single Scrollable Page

All settings on one page with section headers. General at top, System below. Simple, everything visible at a glance. Matches the existing TranscribePage pattern (centered StackPanel, max-width 700).

B

Tabbed Sections

A tab strip at the top: "General" and "System". Each tab shows its own group of settings. Keeps each view short but adds navigation overhead for a relatively small number of settings.

C

Grouped Cards

Single page with settings organized into distinct cards (bordered sections). Each card is a group: "Keybinds", "Microphone", "App Settings", "Sound", "Extras". Visual separation without navigation. Similar to Windows 11 Settings style.
